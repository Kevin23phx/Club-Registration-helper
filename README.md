## Project Description
This console application streamlines the club registration process for student organizations. It collects member information including personal details, membership preferences, and payment information. The system calculates total fees with applicable discounts and displays a comprehensive registration summary. Built using foundational Python concepts including variables, basic data types, input/output operations, and arithmetic expressions.

## How to Run
```bash
python part1.py
```

## Features
- Collects personal and membership information
- Calculates registration fees automatically
- Applies discount for eligible members
- Displays professional registration summary
- Calculates membership validity period

- **Language**: Python 3.x
- **Type**: Console Application
- **Concepts Used**: Variables, data types, input/output, type casting, arithmetic operators
